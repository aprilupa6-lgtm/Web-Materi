Apa itu Web?

Web adalah sistem layanan internet yang memungkinkan pengguna mengakses informasi melalui browser.

Sejarah Singkat Web

  • Diciptakan oleh Tim Berners-Lee (1989)
  • Menggunakan HTTP, URL, dan HTML

Komponen Website

  • Client (Browser)
  • Server
  • Domain & Hosting
