Pertemuan 3 – CSS Dasar

Pendahuluan

Setelah memahami struktur dasar website menggunakan HTML, tahap selanjutnya dalam pengembangan web adalah memperindah tampilan halaman menggunakan CSS. Tanpa CSS, sebuah website hanya akan menampilkan teks dan elemen secara polos sehingga kurang menarik dan kurang nyaman digunakan. Oleh karena itu, CSS memiliki peran penting dalam menciptakan tampilan website yang profesional.

Pengertian CSS

CSS (Cascading Style Sheets) adalah bahasa stylesheet yang digunakan untuk mengatur tampilan dan layout elemen HTML. CSS memungkinkan pengembang untuk mengatur warna, ukuran, jarak, posisi, font, serta berbagai efek visual lainnya pada halaman web.

Fungsi dan Peran CSS

  • Mengatur warna latar belakang dan teks
  • Mengatur ukuran dan jenis huruf
  • Mengatur tata letak (layout) halaman
  • Memberikan efek visual seperti bayangan, animasi, dan transisi
  • Meningkatkan kenyamanan dan pengalaman pengguna

Cara Menyisipkan CSS

CSS dapat ditambahkan ke dalam dokumen HTML dengan tiga cara, yaitu:

  • Inline CSS – Langsung pada tag HTML
  • Internal CSS – Diletakkan di dalam tag <style>
  • External CSS – Menggunakan file .css terpisah

Contoh CSS

body {
  background-color: #f4f6f9;
  font-family: Arial, sans-serif;
}

h1 {
  color: #1f3c88;
}

Konsep Selector

Selector adalah bagian CSS yang digunakan untuk memilih elemen HTML yang akan diberikan style. Beberapa selector yang sering digunakan:

  • Selector tag (h1, p, div)
  • Selector class (.card, .sidebar)
  • Selector id (#header)

Box Model CSS

Box Model adalah konsep dasar CSS yang menjelaskan bahwa setiap elemen HTML dianggap sebagai sebuah kotak yang terdiri dari margin, border, padding, dan content. Pemahaman box model sangat penting dalam mengatur jarak dan ukuran elemen pada halaman web.

Manfaat Mempelajari CSS

  • Membuat website lebih menarik dan profesional
  • Meningkatkan user experience
  • Mempermudah pengaturan layout
  • Mendukung desain responsif

Kesimpulan

CSS merupakan komponen penting dalam pemrograman web yang berfungsi untuk mengatur tampilan dan layout halaman. Dengan memahami CSS, pengembang dapat menciptakan website yang lebih indah, nyaman, dan profesional.
