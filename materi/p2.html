Pertemuan 2 – HTML Dasar

Pengertian HTML

HTML (HyperText Markup Language) adalah bahasa markup standar yang digunakan untuk membuat dan menyusun struktur halaman web. HTML bukanlah bahasa pemrograman, melainkan bahasa penanda (markup language) yang berfungsi untuk memberi struktur dan makna pada konten, seperti teks, gambar, tabel, formulir, serta elemen multimedia.

Setiap halaman website yang kita akses melalui browser pada dasarnya dibangun dari dokumen HTML. Browser akan membaca file HTML kemudian menampilkannya dalam bentuk visual kepada pengguna.

Struktur Dasar Dokumen HTML

Setiap dokumen HTML memiliki struktur dasar sebagai berikut:

<!DOCTYPE html>
<html>
<head>
  <title>Judul Halaman</title>
</head>
<body>
  Konten Website
</body>
</html>

Tag <!DOCTYPE> digunakan untuk mendeklarasikan jenis dokumen HTML yang digunakan. Tag <html> merupakan elemen utama yang membungkus seluruh isi halaman. Tag <head> berisi informasi meta, sedangkan <body> berisi konten yang tampil di browser.

Elemen dan Tag HTML

HTML terdiri dari berbagai elemen yang ditulis menggunakan tag. Beberapa tag penting antara lain:

  • <h1> – <h6> : Digunakan untuk judul
  • <p> : Paragraf
  • <a> : Link
  • <img> : Gambar
  • <ul>, <ol>, <li> : Daftar
  • <table> : Tabel
  • <form> : Form input

Contoh Penggunaan Tag

<h1>Website Pertama Saya</h1>
<p>Ini adalah paragraf pertama saya.</p>
<a href="https://google.com">Kunjungi Google</a>

Manfaat Mempelajari HTML

  • Menjadi dasar dalam pengembangan website
  • Memahami struktur dan kerangka web
  • Mempermudah mempelajari CSS dan JavaScript
  • Digunakan pada semua framework web

Kesimpulan

HTML merupakan pondasi utama dalam pengembangan web. Dengan memahami HTML, kita dapat membuat struktur halaman web yang rapi, terorganisir, dan siap dikembangkan lebih lanjut menggunakan CSS dan JavaScript.
