Pertemuan 4 – JavaScript Dasar

Pendahuluan

Pada tahap pengembangan website, JavaScript memegang peranan penting dalam memberikan interaksi dan dinamika pada halaman web. Dengan JavaScript, sebuah website tidak hanya menampilkan informasi secara statis, tetapi juga mampu merespons tindakan pengguna seperti klik, input, dan perubahan data secara langsung di browser.

Pengertian JavaScript

JavaScript adalah bahasa pemrograman yang berjalan di sisi klien (client-side) yang digunakan untuk membuat halaman web menjadi interaktif dan dinamis. JavaScript dapat mengontrol elemen HTML, mengubah CSS, memproses input pengguna, serta menampilkan pesan atau peringatan secara langsung.

Fungsi JavaScript dalam Website

  • Membuat efek interaktif
  • Memvalidasi form input
  • Menampilkan notifikasi dan alert
  • Mengubah konten halaman tanpa reload
  • Meningkatkan pengalaman pengguna

Contoh Kode JavaScript

function tampilPesan() {
  alert("Selamat datang di Website Pemrograman Web!");
}

Penggunaan JavaScript pada HTML

JavaScript dapat ditambahkan pada HTML dengan dua cara, yaitu:

  • Internal JavaScript (di dalam tag <script>)
  • External JavaScript (file .js terpisah)

Contoh Penggunaan

<button onclick="tampilPesan()">Klik Saya</button>

Keunggulan JavaScript

  • Dapat dijalankan langsung di browser
  • Tidak memerlukan server tambahan
  • Mudah dipelajari
  • Mendukung berbagai library dan framework

Kesimpulan

JavaScript merupakan komponen penting dalam pemrograman web yang membuat website menjadi lebih hidup dan interaktif. Dengan menguasai JavaScript, pengembang dapat menciptakan website yang modern, responsif, dan user-friendly.
