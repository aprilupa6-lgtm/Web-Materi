Pertemuan 1 – Pengantar Web

Pendahuluan

Perkembangan teknologi informasi telah membawa perubahan besar dalam berbagai bidang kehidupan, salah satunya melalui hadirnya internet dan website. Website menjadi media utama dalam penyampaian informasi, komunikasi, serta layanan digital yang dapat diakses oleh siapa saja dan dari mana saja. Oleh karena itu, pemahaman mengenai konsep dasar web sangat penting bagi mahasiswa yang mempelajari pemrograman web.

Pengertian Website

Website adalah kumpulan halaman web yang saling terhubung dan dapat diakses melalui jaringan internet menggunakan browser seperti Google Chrome, Mozilla Firefox, dan Microsoft Edge. Setiap website memiliki alamat unik yang disebut URL (Uniform Resource Locator) yang berfungsi sebagai identitas website di internet.

Website terdiri dari berbagai elemen seperti teks, gambar, video, audio, serta komponen interaktif yang bertujuan untuk menyampaikan informasi atau memberikan layanan tertentu kepada pengguna.

Komponen Utama Website

Sebuah website dibangun dari tiga komponen utama, yaitu:

  • HTML – Digunakan untuk menyusun struktur halaman web
  • CSS – Digunakan untuk mengatur tampilan dan layout
  • JavaScript – Digunakan untuk memberikan interaksi pada website

Cara Kerja Website

Ketika pengguna mengetikkan alamat website pada browser, browser akan mengirimkan permintaan (request) ke server tempat website disimpan. Server kemudian mengirimkan kembali file HTML, CSS, dan JavaScript ke browser. Browser akan memproses file tersebut dan menampilkannya dalam bentuk halaman web yang dapat dilihat oleh pengguna.

Jenis-Jenis Website

  • Website Statis – Konten tetap dan jarang berubah
  • Website Dinamis – Konten dapat berubah dan menggunakan database
  • Website E-Commerce – Digunakan untuk jual beli online
  • Website Portal – Menyediakan berbagai informasi terpusat
  • Website Blog – Digunakan untuk publikasi artikel

Peran Pemrograman Web

Pemrograman web merupakan proses pembuatan website dengan menggunakan bahasa pemrograman dan teknologi web. Dengan pemrograman web, kita dapat membuat website yang interaktif, dinamis, serta memiliki berbagai fitur seperti formulir, login, sistem komentar, dan lain sebagainya.

Manfaat Mempelajari Pemrograman Web

  • Memahami cara kerja internet dan website
  • Mampu membuat website sendiri
  • Meningkatkan peluang kerja di bidang IT
  • Mendukung bisnis digital dan wirausaha

Kesimpulan

Website merupakan sarana utama dalam dunia digital saat ini. Dengan memahami konsep dasar web, mahasiswa akan memiliki fondasi yang kuat untuk mempelajari HTML, CSS, dan JavaScript pada pertemuan-pertemuan berikutnya.
